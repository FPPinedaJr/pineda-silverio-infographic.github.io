How to Convert Hexadecimal to Octal

Step 1.

Understand how the conversion work. To convert from hexadecimal to octal, it is necessary to:

  1. convert the hexadecimal to binary;
  2. convert the binary to octal.

Step 2.

Convert it to binary. First find the binary equivalence of each hexadecimal digit and just simply reformat them. for more details, click here

Note: First, convert the hexadecimal letters to numbers as shown in the table below.

Numerical Value Hexadecimal Value
10 A
11 B
12 C
13 D
14 E
15 F

Step 3.

Group the binary into 3. After the binary is reformatted, you have to group the binary into 3 digits to be converted to octal. If the last group don't have 3 digits, you can add zeros on the left to make it easier for you. Using the example above:

  • 2A516 = 001010100101 = 10101001012
  • 10101001012 = (1)(010)(100)(101) = (001)(010)(100)(101)

Step 4.

Convert the binary to octal. You have to find the octal of each binary group. for more details, click here

In the example above:

  • 001=1
  • 010=2
  • 100=4
  • 101=5

Step 5.

Write the answer along with its correct base subscript. Now, all you have to do to show that you are using a octal answer that must be operating in powers of 8, so write 1245 with base 16.
