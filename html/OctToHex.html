How to Convert Octal to Hexadecimal

Step 1.

Understand how the conversion work. To convert from octal to hexadecimal, it is necessary to:

  1. convert the octal to binary;
  2. convert the binary to hexadecimal.

Step 2.

Convert it to binary. First find the binary equivalence of each octal digit and just simply reformat them. for more details, click here

Step 3.

Group the binary into 4. After the binary is reformatted, you have to group the binary into 4 digits to be converted to hexadecimal. If the last group don't have 4 digits, you can add zeros on the left to make it easier for you. Using the example above:

  • 1758 = 001111101 = 11111012
  • 11111012 = (111)(1101) = (0111)(1101)

Step 4.

Convert the binary to hexadecimal. You have to find the hexadecimal of each binary group. for more details, click here

In the example above:

  • 0111=7
  • 1101=13=D

Step 5.

Write the answer along with its correct base subscript. Now, all you have to do to show that you are using a hexadecimal answer that must be operating in powers of 16, so write 7D with base 16.
