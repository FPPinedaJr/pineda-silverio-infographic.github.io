How to Convert from Binary to Hexadecimal

Step 1.

Starting from the right, divide your string of binary integers into groups of four. Four binary digits are converted into one hexadecimal unit. Therefore, you must first divide the binary number into groups of four, starting on the right, before you can convert it. For instance:

  • Convert 1110001111011 into a hexadecimal number.
  • 1110001111011 = (1)(1100)(0111)(1011)

Step 2.

If the initial number is not four digits, add more zeros to the front.Although the zeros won't change the conversion, they will make it simpler to picture. Always keep in mind that you want all sets of 4-digit binary numbers.

  • Convert 1110001111011 into a hexadecimal number.
  • 1110001111011 = (1)(1100)(0111)(1011)
  • (1)(1100)(0111)(1011) = (0001)(1100)(0111)(1011)

Step 3.

Convert one group of four digits at a time. Each binary set will need to be converted separately, so add them out on your paper to make things simplier. Work on converting each string of four digit binary to its corresponding numerical value. As an illustration:

  • 0001=0+0+0+1=1
  • 1100=8+4+0+0=12
  • 0111=0+4+2+1=7
  • 1011=8+0+2+1=11

Step 4. Create your hexadecimal number. After converting each of the 4-digit components, first convert the numbers that are 10 and above to letters as shown in the table below then just simply join them all up to get the hexadecimal number.

Numerical Value Hexadecimal Value
10 A
11 B
12 C
13 D
14 E
15 F

Hence, using the example before:

  • 0001=1
  • 1100=12=C
  • 0111=7
  • 1011=11=B
  • (0001)(1100)(0111)(1011)=1C7B

Step 5.

Write the answer along with its correct base subscript. Now, all you have to do to show that you are using a hexadecimal answer that must be operating in powers of 16, so write 1C7B with base 16.
