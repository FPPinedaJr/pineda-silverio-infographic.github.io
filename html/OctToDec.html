How to Convert Octal to Decimal

Step 1.

Write down the octal number, then list the powers of 8 in order from right to left. Write the octal number first, next list the powers of 8 going from right to left. Start with 80 then keep increasing the power until it is the same number of digits as the octal numbers. In the example, there are 3 digits thus the list should have 64, 8, 1.

Step 2.

Multiply the list number with it's corresponding octal digit. Each place value of the octal number should be multiplied by the matching power of eight to get the decimal equivalent. In this example:

  • 6 * 64 = 384
  • 5 * 8 = 40
  • 3 * 1 = 3

Step 3.

Sum up the computed values. Add the numbvrs that are listed below the line. In this case add the 384 + 40 + 3 which will yield a sum of 427. This is the octal number 653's decimal conversion.

Step 4.

Write the answer along with its correct base subscript. Now, all you have to do to show that you are using a decimal answer that must be operating in powers of 10, so write 427 with base 10.
