How to Convert Decimal to Binary

Step 1.

Write the given. To convert the decimal number 15610 to binary as an example, in a "long division" sign that is upside-down, write the dividend as the given decimal number. Outside the curve of the division sign, write the divisor as the base of the destination system, in this case, "2" for binary.

  • This approach relies simply on division by two, making it considerably simpler for beginners to understand when depicted on paper.
  • Write the number of the base system that you are using as a subscript to each number to avoid confusion before and after conversion. The binary equivalent will have a subscript of 2, while the decimal number will have a subscript of 10.

Step 2.

Divide. Write the quotient below the division symbol and the remainder(0 or 1) to the right of the quotient.

  • Since we are dividing by 2, the remainder will be either 0 or 1.
  • In this case, 156 / 2 is equal to 78 remainder 0.

Step 3.

Divide again till you get to 0. As you descend, divide each fresh quotient by 2 again, then write the results below the division symbol and the remainder to the right. When the quotient is 0, stop.

Step 4.

The new binary number in writing. Read the series of remainders from the bottom up, starting with the lowest one. 10011100 for this example. This represents the decimal number 156 in binary. Alternately, write this way: 15610 = 100111002.
